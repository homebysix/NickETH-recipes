Mozilla Thunderbird ESR x64 60.0.0 ML (Multi-Language)

Email Client. Outlook alternative

Tested mit Windows 10 + 11

See the release notes for changes.
Mozilla Release Notes for TB 145.0.0: https://www.thunderbird.net/en-US/thunderbird/145.0/releasenotes/

Start the installation with the following command line:
"ThunderbirdESR_66.0.2_ML.exe" /DesktopShortcut=false /TaskbarShortcut=false /MaintenanceService=false

Features in this version:
- Multi-Language enu, deu, fra, ita, rum.
- Spellchecker for enu, deu, fra, ita.
- Thunderbird language follows the system language automatically (with multilanguge-pack too).
- Almost all settings can be set through group policies. See Thunderbird policies for mor information.
- With romansh interface

  Remark: Group Policy and CAL/DAV are now natively implemented in TB.
